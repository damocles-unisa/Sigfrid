# Sigfrid

This repository contains the supplementary material for the paper "LLM-Powered Scene Interaction Graphs for Interference Detection in Trigger-Action Rules" submitted to IEEE Internet of Things Journal.

This material comprises the source codes useful for the repeatability of the experiments.


# Overview

SIGFRID, denoted as <b>S</b>cene <b>I<b/>nteraction <b>G<b/>raphs <b>f</b>or <b>I</b>nterference <b>D<b/>etection, operates as an unsupervised, platform-independent utility. Driven by Large Language Models (LLMs), it autonomously discerns diverse interference manifestations arising from unanticipated rule interactions. Leveraging LLMs, SIGFRID deduces causal relations among events in the smart home domain, thereby formulating an exhaustive Scene Interaction Graph (SIG) that encapsulates interconnections among rules. The proficient exploration of SIG facilitates the extraction of interference instances.
